# Installation

`sparse_grid` can be installed from PyPI and Git.

## PyPI

### `pyproject.toml`

```toml
[project]
dependencies = [
	"sparse_grid"
]
```

### `requirements.txt`

```text
sparse_grid
```

## pip

```bash
pip install --upgrade sparse_grid
```

## pipenv

```bash
pipenv install sparse_grid
```

## uv

### Add to a uv project

```bash
uv add sparse_grid
```

### Install into the current uv environment

```bash
uv pip install sparse_grid
```

## poetry

```bash
poetry add sparse_grid
```

## pdm

```bash
pdm add sparse_grid
```

## hatch

```bash
hatch add sparse_grid
```

## GitHub

Install the latest code from the repository:

```bash
pip install --upgrade "git+https://github.com/eggzec/sparse_grid.git#egg=sparse_grid"
```

## Dependencies

- Python 3.10+
- No external runtime dependencies
